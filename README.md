# WPM Hero

## Frontend

### Install

```
  npm install
```

### Running and testing
```
  npx vite

  npx vitest
```
## Backend

### Install

Ensure you have SDKMAN installed and that you are in the backend folder to make use of the .sdkmanrc file.

```
  ./mnvw install
```

### Running and testing

```
  ./mnvw spring-boot:run

  ./mnvw test
```
